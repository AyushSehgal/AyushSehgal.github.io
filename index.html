About

Hey, I'm Ayush. Born and raised in Bangkok, Thailand. I graduated from Ruamrudee International School with an IB Diploma in 2018. I completed my Undergraduate Degree in Computer Science and Data Science at UC Berkeley in Spring 2022. I am currently a Software Engineer at Electronic Arts and am part of the social team working to integrate friends/groups based social features in games.

Post-Grad Projects

Spotify Calendar (in-progress)

This project involves using the Spotify Web API to display a history of listening patterns in a calendar-like format. The API is being used here with OAuth to get user-specific data to be displayed. I started this project primarily to learn the Spotify API and also for myself and my friends to see how our music tastes have grown with us.

Expenses Automated Calculator (in-progress) Github

This is a python app that takes a bank statement pdf as input and outputs a data about spending categories and suggests more effective spending habits. I use an app for this but I needed better control over it and it was taking too long to manually tag each expense or to manually read through these bank statements so this tool makes my life easier as it automates that and reads the pdf for me.

PC/Server Build

After spending many years watching Linus Tech Tips videos and bitwit on Youtube, I finally made the decision to build my own desktop computer. This was a great project, I learned the ins and outs of how to set up a computer and built mine as a gaming device as well as a NAS-like server to store all my files and to use as a power machine to run large programs and to fund my interests in ML and high-performance computing.

F1 Data Analyzer (in-progress) Github

This passion project came as a result of my anticipation of the 2023 F1 season, I wanted a way to figure out who has the chance of clinching the championship and to see how a driver's season has been going with easily producible data. I found myself doing the same calculations throughout the season so this tool helped me automate that process in a fun and easy-to-view way. I build this primarily in python as a script and use web-scraping libraries like bs4 to scrape latest data off of the F1 website.
